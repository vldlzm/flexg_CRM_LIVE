# 레이아웃 규칙 (절대 변경 금지)

## 패널 구조
이 프로젝트의 레이아웃은 정확히 3개의 패널로 고정되어 있습니다.
절대로 패널을 추가하거나 중복 생성하지 마세요.

- **좌측 패널 1개**: 화면목록 (`src/app/page.tsx` 내 `<aside>`)
- **중앙 패널 1개**: 화면 미리보기 (`<iframe>`)
- **우측 패널 1개**: 화면설명 (`src/app/page.tsx` 내 `<aside>`)

## 금지 사항
- 새로운 패널 컴포넌트 생성 금지
- 기존 패널 래핑 또는 중첩 금지

## iframe 내 GNB/LNB 숨김 방식
- `window.self !== window.top` 으로 iframe 여부 감지
- URL 파라미터(`?embed=1`) 방식은 내부 링크 클릭 시 유실되므로 단독 사용 금지
- 인덱스 페이지(`/`)는 iframe 내 로드 시 `null` 반환
